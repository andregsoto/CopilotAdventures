                        CORE FUNCTIONALITY
================================================================================

âœ¨ EchoChamber Class Features:

  1. validateArithmeticProgression(sequence)
     - Validates if a sequence follows an arithmetic progression
     - Checks for valid array input and numeric elements
     - Returns difference value and validation status

  2. predictNextNumber(sequence)
     - Predicts the next number in an arithmetic sequence
     - Stores memory of the echo with timestamp
     - Returns detailed prediction results

  3. getMemories()
     - Returns all stored echo memories
     - Each memory contains sequence, prediction, difference, timestamp

  4. clearMemories()
     - Clears all stored memories
     - Resets echo counter

  5. getStatistics()
     - Returns statistics about all echoes
     - Includes total echoes, average difference, first/last echo
